# Sales Performance Intelligence Dashboard

## Project Overview

This project analyzes sales performance, customer segments, payment methods, and revenue trends using Power BI. The dashboard provides business insights to support data-driven decision-making.

## Tools Used

* Power BI
* Microsoft Excel

## Dashboard Features

* Total Sales
* Total Orders
* Total Customers
* Month-over-Month Sales Growth
* Average Order Value
* Net Revenue
* Sales by Category
* Sales by Payment Mode
* Customer Segment Analysis
* Product Performance Analysis
* Delivery Performance Analysis
* Geographic Sales Distribution

## Key Insights

* Electronics generated the highest sales revenue.
* Cash was the most commonly used payment method.
* Corporate and Enterprise customers contributed significantly to revenue.
* Customer segmentation helped identify high-value customer groups.
* Geographic analysis revealed top-performing sales regions.
